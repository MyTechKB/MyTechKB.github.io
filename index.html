chmod Hierarchy

Linux - Problem: How to set chmod for a folder and all of its subfolders and files in Linux Ubuntu Terminal ? Linux - Solution 1: chmod -R 755 will set this as permissions to all files and folders in the tree. You can use the find command. For example: # To change all the directories to 755 (drwxr-xr-x): find /opt/lampp/htdocs -type d -exec chmod 755 {} \; # To change all the files to 644 (-rw-r--r--): find /opt/lampp/htdocs -type f -exec chmod 644 {} \;
