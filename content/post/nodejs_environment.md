## To create a new package.json
{{< highlight bash >}}
npm init -y
{{< / highlight >}}

## To install a package
{{< highlight bash >}}
npm i <package-name>
sudo npm i -g nodemon
{{< / highlight >}}

## To install a local package.json
{{< highlight bash >}}
npm install
{{< / highlight >}}

## To run node file
{{< highlight bash >}}
node app.js
{{< / highlight >}}

## To use nodemon to auto-restart server
{{< highlight bash >}}
nodemon app.js
{{< / highlight >}}
